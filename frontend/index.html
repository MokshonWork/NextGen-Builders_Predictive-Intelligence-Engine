Model Rationalization: "Why This Forecast?"

Explaining Google TimesFM structural predictions

Historical Patterns

The model analyzes the input series for baseline trends, cyclic recurrence, and structural breaks. It registers underlying patterns to inform standard seasonal weights.

Trend Direction

TimesFM predicts the slope and velocity of the series over the forecast horizon. Extrapolations are calculated with zero-shot weights based on millions of pre-trained datasets.

Seasonal Behavior

Identifies cyclical behaviors in the dataset. Autoregressive weights account for weekly offsets and month-end fluctuations based on historical frequency matching.

Scenario Impact

Simulates simulated overrides to the baseline forecasts. The scenario simulator scales forecasted levels according to demand overrides while maintaining underlying variance bounds.

FutureSight AI System Architecture

Frontend Framework
Streamlit
Provides the robust Python web environment, running an iframe-based premium React-like SPA component for absolute style control.
AI Workflow Orchestrator
LangGraph
Defines step-by-step stateful workflows (`Validation -> Cleaning -> Forecast -> Insights`) enabling granular, agentic tracking.
Forecasting Foundation
Google TimesFM (200M)
State-of-the-art transformer architecture trained on 100B+ time-series points, delivering highly accurate zero-shot daily predictions.
Visualizations
Plotly.js
Interactive, GPU-accelerated client-side plotting supporting dynamic dark/light configuration, confidence bounds, and zoom grids.
Data Processing
Pandas + NumPy
Fast, native matrix computation pipelines for outlier capping, linear interpolation, and scenario overrides.
